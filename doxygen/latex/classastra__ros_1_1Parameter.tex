class astra\+\_\+ros\+::\+Parameter$<$ T $>$}

A \char`\"{}live\char`\"{} value that can be changed at runtime. Changes are broadcast to a listener. 

Parameters can be copied cheaply. They are backed by {\ttfamily \mbox{\hyperlink{classastra__ros_1_1ParameterImpl}{Parameter\+Impl}}}, which implements the actual logic.
